## 주제: **PlantyNote(식물 관리 앱)**

---

## 프로젝트 설명  
**PlantyNote**는 사용자들이 자신의 식물을 효과적으로 돌볼 수 있도록 돕는 **모바일 애플리케이션**입니다.  
사용자는 **식물 정보 관리, 메모 기능** 등을 통해 자신의 식물과 더욱 친밀해질 수 있습니다.  
식물 관리가 어렵다고 느꼈다면, 이제 **PlantyNote**와 함께라면 걱정할 필요가 없어요! 🌼
